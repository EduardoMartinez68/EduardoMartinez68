What is PLUS?

PLUS ERP is an open-source project because I deeply believe in the power of community, collaboration, and shared learning 🌎💻.

Free and open-source software allows us to:

  • ✅ Learn from each other
  • ✅ Adapt tools to our own needs
  • ✅ Improve the software together
  • ✅ Democratize access to quality technology

PLUS ERP is free for any company, developer, or curious mind to study, modify, and use. The only thing I ask in return is that you follow a few small conditions, so the community can grow in a healthy and respectful way:

🔖 Attribution

If you use PLUS ERP or any part of its code in your projects, please give credit to the original creator:

Created by: Martínez Ortiz Eduardo Antonio (Mexico) 🙋🏻‍♂️

🌱 Share Your Improvements

If you make any cool modifications or improvements, it would be awesome if you shared them back with the community via Pull Requests or by publishing your fork. That way, we all learn and the project keeps getting better 🚀.

🔓 Don’t Close the Code

You’re not allowed to take this engine and turn it into a closed or proprietary product. The idea is for it to remain open forever — accessible and transparent.

🤝 Respect Within the Community

If a community forms around PLUS ERP (forums, Discord, GitHub Issues), please always participate with respect, openness, and good vibes.

👉 Why Open Source?

Because the magic of open source lies in collaboration. By sharing your knowledge and your work, you help others improve the software, learn, and build amazing solutions we might’ve never imagined.

PLUS ERP aims to inspire other Mexican and Latin American developers to create tools tailored to our own context, language, businesses, and needs 🎉.

I've always believed that our purpose as programmers is to create Technology with a touch of magic ❤️
