📦 Technical Installation

Get PLUS ERP up and running

This guide will help you install PLUS ERP on your local machine.

🔧 Requirements

  • 🐍 Python (3.10 or higher recommended)
  • 🐘 PostgreSQL (Ensure it's running locally or accessible via network)
  • 🌿 Git
  • 📦 pip (Python package manager)
  • 📁 npm (for frontend assets if needed in future versions)

📥 Installation Steps

  1. Clone the project from GitHub: 
    git clone https://github.com/EduardoMartinez68/PLUS_ENGINE_ERP
  2. Navigate into the project folder: 
    cd plus-erp
  3. Install Python dependencies: 
    pip install -r requirements.txt
  4. (Optional) Install Node.js dependencies (if needed for future frontend assets): 
    npm install
  5. Start the server: 
    python manage.py runserver

You should now be able to open your browser and access the ERP at:

http://localhost:8000

👨‍💻 Technical Overview

PLUS ERP is a modular system written in Python with a clean and minimal frontend. Below is a quick tech breakdown:

  • 🎨 Frontend: HTML + CSS (no Bootstrap)
  • 🧠 Backend: Python
  • 📊 Database: PostgreSQL
  • 🧱 Architecture: Lightweight MVC
  • 🎨 CSS Style: Custom styling using prefixes like sub-menu-app- to avoid class name conflicts

This modular architecture makes PLUS ERP easy to maintain, extend, and customize for your own business needs.
