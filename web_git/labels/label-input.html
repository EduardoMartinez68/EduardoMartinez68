To create an input field in the ERP, use <input-field>

The custom component <input-field> makes it easier to add labeled and translatable input fields.

ðŸ“Œ Examples

You can choose to use static text or translation keys:


  ------when you not need translate the container--------
  <input-field label="Amount that buy to the company *"
    placeholder="The amount of this item that you buy from the supplier.">
  </input-field>

  ------when you need translate the container. Add the key of the dictionary--------
  <input-field label="label-search-home"
    placeholder="placeholder-input-search-home">
  </input-field>

This approach gives you full flexibility to use static strings or translated keys depending on your context.
Make sure that all keys used in label and placeholder exist in your translate.json file to avoid missing translations.
