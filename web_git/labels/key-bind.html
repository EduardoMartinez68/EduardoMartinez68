⌨ key-bind

Creating Keyboard Shortcuts with <key-bind>

The <key-bind> custom element allows you to define global keyboard shortcuts that trigger specific JavaScript functions. This is ideal for enhancing productivity and user experience in your ERP system or application.

Usage:


<!-- Basic example: Run a function when Shift + V is pressed -->
<key-bind combo="Shift+V" action="openMenu"></key-bind>

This element works silently in the background and does not interfere with your visual layout. Once declared, it listens for the key combination and runs the specified function defined in window.

---

Attributes:

Attribute Description Required
combo The key combination to listen for (e.g. Shift+V, Ctrl+Enter, Alt+1). ✅ Yes
action The name of the JavaScript function in window to execute when the combo is triggered. ✅ Yes
---

JavaScript Function Example:

Make sure the function you reference exists globally (attached to window):


<script>
function openMenu(){
    alert("Menu opened!");
  }

function createNewClient(){
    console.log("Creating new client...");
  }
</script>
---

Multiple Key Bindings:

You can define as many key-bind shortcuts as you need:


<key-bind combo="Shift+V" action="openMenu"></key-bind>
<key-bind combo="Ctrl+N" action="createNewClient"></key-bind>
---

Important Notes:

  • The combo is case-insensitive and trims extra spaces.
  • If the specified function does not exist in window, a warning will be shown in the console.
  • Each <key-bind> automatically hides itself with display: none.
---

Supported Keys:

Any key recognized by e.key in JavaScript can be used, such as:

  • Shift, Control / Ctrl, Alt, Meta (Command key)
  • Letters: A to Z
  • Numbers: 0 to 9
  • Special keys: Enter, Tab, Escape, ArrowUp, etc.
---

Advanced Example:


<key-bind combo="Shift+V" action="openMenu"></key-bind>
<key-bind combo="Ctrl+N" action="createNewClient"></key-bind>

function openMenu(){
    alert("Menu opened!");
  }

function createNewClient(){
    console.log("Creating new client...");
  }
---

This system lets you define powerful keyboard workflows without cluttering your HTML with event listeners or JavaScript code. Each shortcut becomes declarative and easy to manage.
