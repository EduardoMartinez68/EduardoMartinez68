📅 <plus-select-date> Custom Element

👉 What Is <plus-select-date> and What Is It Used For?

<plus-select-date> is a custom HTML element that provides a date range selector with predefined options (Today, Last 7 Days, Current Month, Last Month, Custom) and a modern user interface.

✅ Use it when: you need to filter data, generate reports, or select date intervals in a user-friendly and reusable way.

---

⚙️ How to Use It in HTML


<plus-select-date 
  name-start="start_date" 
  name-end="end_date" 
  label="Date Range" 
  required 
></plus-select-date>

✅ Attributes:

  • name-start: Name/id of the hidden input for the start date.
  • name-end: Name/id of the hidden input for the end date.
  • label: Optional label shown above the input.
  • t: Optional translation key (uses window.translate_text()).
  • required: Optional. Adds a * mark to indicate the field is mandatory.
---

📦 Requirements

  • ✅ Browser with Shadow DOM support
  • ✅ Global window.translate_text() and window.t() translation functions
  • ✅ `Flaticon UIcons` (calendar icons)
  • ✅ CSS variable --primary for styling (set in your main CSS)
---

🎨 Features

  • 🌐 Language-aware labels (uses browser language for date formatting)
  • 📌 Predefined ranges:
    • Today
    • Last 7 Days
    • This Month
    • Last Month
    • Custom Range (manual date pickers)
  • 📥 Hidden inputs store start and end dates in `YYYY-MM-DD` format
  • ✨ Smooth UX with dropdown and live updates
---

🧠 How It Works

  1. The component builds a calendar icon button and dropdown panel inside its shadow DOM.
  2. When a range is selected, the start/end inputs are updated automatically.
  3. In "Custom" mode, two native `` fields allow manual selection.
  4. The result is shown as a formatted string (e.g., “August 10, 2025 - August 13, 2025”).
---

<input type="hidden" name="start_date" id="start_date">
<input type="hidden" name="end_date" id="end_date">
---

📌 Example Behavior

- User clicks the calendar field. - A dropdown appears with quick date options. - Selecting "Last 7 Days" auto-fills the hidden inputs. - Choosing "Custom" lets user pick start/end dates manually. - Clicking “✅ Accept” saves the range and closes the popup. - Clicking “❌ Cancel” closes the popup with no changes. ---
