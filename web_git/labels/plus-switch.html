ğŸ§  How It Works

  1. It reads the text or t attribute and passes it to the window.translate_text() function.
  2. It checks the checked attribute to decide whether the switch should be initially ON or OFF.

  4. The final DOM is rendered using native HTML elements with appropriate CSS classes.
