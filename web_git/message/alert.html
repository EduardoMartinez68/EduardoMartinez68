JavaScript function to display popup alert windows in the interface. It allows showing different types of messages to the user: information, success, error, question, etc.

show_alert() dynamically updates a custom alert window in the HTML, changing its text, icons, colors, and buttons based on the type of message you want to display.

    function show_alert(
        type,           // (string) Alert type: 'info', 'success', 'alert', 'question'
        title,          // (string) Title to display in the alert
        description,    // (string) Descriptive text of the message
        readmoreText    // (string, optional) Additional text (e.g., technical errors)
    )

üîî show_notification()

JavaScript function to display floating toast-style notifications. Ideal for quickly informing the user about the result of an action: success, error, warning, etc.

show_notification() dynamically creates a small notification card that appears on the interface for a few seconds, with custom icons and styles depending on the message type.

function show_notification(
    type = 'info',    // (string) Type: 'success', 'error', 'warning', 'info'
    message = '',     // (string) Notification text
    duration = 4000   // (int) Time in ms (default: 4000)
)
